# MiniTV

## 제품 구매 후, 설치방법
2.8인치 액정을 동작시키려면 [여기](https://www.waveshare.com/wiki/2.8inch_DPI_LCD)에 나와 있는 지침을 따라 라즈비안 설치부터 화면 회전까지 순서대로 진행하세요. 스피커는 GPIO 18, GPIO 19에 연결됩니다. GPIO 18은 2.8인치 디스플레이가 사용하므로 점퍼 커넥터를 제거하세요. 배터리와 라즈베리파이를 연결합니다.
